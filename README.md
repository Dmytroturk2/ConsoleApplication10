# Weather Recorder System

Простая и эффективная система для отслеживания и анализа метеорологических данных на C++.

## 📋 Описание

**Weather Recorder System** — это консольное приложение, которое позволяет:
- Создавать записи о погодных условиях с информацией о дате, температуре, влажности и осадках
- Управлять журналом погодных данных
- Анализировать метеорологические показатели (средняя, максимальная и минимальная температура)
- Отслеживать дни с осадками
- Сравнивать погодные условия между разными днями

## 🎯 Основные функции

### Класс WeatherRecorder
Представляет одну запись о погодных условиях на определённую дату.

**Методы:**
- `getDate()` — получить дату записи
- `getTemperature()` — получить температуру
- `getHumidity()` — получить влажность (%)
- `getPrecipitation()` — получить количество осадков (мм)
- `hasPrecipitation()` — проверить наличие осадков
- `isWarmerThan(other)` — сравнить температуру с другой записью
- `isSameDay(other)` — проверить, относятся ли две записи к одному дню
- `display()` — вывести информацию о записи

**Конструкторы:**
- Конструктор по умолчанию — инициализирует пустую запись
- Конструктор с параметрами — создаёт запись с указанными данными

```cpp
WeatherRecorder rec1("2024-01-10", 5.5, 65, 2.3);  // с параметрами
WeatherRecorder rec2;                               // по умолчанию
```

### Класс WeatherLog
Управляет журналом метеорологических записей.

**Методы:**
- `addRecord(record)` — добавить запись в журнал
- `getCount()` — получить количество записей
- `getAverageTemperature()` — вычислить среднюю температуру
- `getHottestDay()` — найти день с наивысшей температурой
- `getColdestDay()` — найти день с наименьшей температурой
- `RainyDays()` — получить список дней с осадками
- `printAll()` — вывести весь журнал

## 🚀 Быстрый старт

### Требования
- C++11 или выше
- Любой C++ компилятор (Visual Studio, GCC, Clang и т.д.)

### Компиляция и запуск

```bash
# Компиляция
g++ -std=c++11 ConsoleApplication10.cpp WeatherRecorder.cpp -o weather_recorder

# Запуск
./weather_recorder
```

### Пример использования

```cpp
#include "WeatherRecorder.h"

int main() {
    // Создание записей
    WeatherRecorder rec1("2024-01-10", 5.5, 65, 2.3);
    WeatherRecorder rec2("2024-01-11", 3.2, 70, 0.0);

    // Создание журнала
    WeatherLog log;
    log.addRecord(rec1);
    log.addRecord(rec2);

    // Анализ данных
    cout << "Average temp: " << log.getAverageTemperature() << "°C" << endl;
    cout << "Hottest day: " << log.getHottestDay() << endl;
    cout << "Rainy days: " << log.RainyDays() << endl;

    // Сравнение записей
    if (rec1.isWarmerThan(rec2)) {
        cout << "Record 1 is warmer than record 2" << endl;
    }

    // Вывод всех записей
    log.printAll();

    return 0;
}
```

Каждая запись о погоде содержит:
- **Дата** (строка, например "2024-01-10")
- **Температура** (double, в °C)
- **Влажность** (int, в %)
- **Осадки** (double, в мм)
